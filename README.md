PREP: THEME IMPORT
1. Open Shopify Admin
2. Open Sales Channels on the left panel
3. Open Themes
4. Import Theme
5. Choose Connect From GitHub
6. Choose correct theme repo

PREP: ADD SECTION TO THEME
1. Under Theme Library, find the dev theme & click Edit Theme
2. Under Template, click Add Section
3. Find and add GOODR5 Pop Up section

AFTER BUILD
1. Enter the dev theme editor
2. Click on GOODR5 Pop Up
3. Unclick Enable Pop Up (This will disable the pop up until go live date)

GO LIVE/LAUNCH DATE
1. Enter theme editor
2. Click the 3 dots to open settings
3. Preview the new template to verify site functionality
4. Click Publish (this will import new theme), verify site functionality once again
5. Enable Pop Up in theme editor, verify site functionality once last time

POST
1. Monitor and review event logs for status reports

ROLLBACK
* Low Impact: Disable Pop Up, prepare findings
* High Impact: Publish previous theme, prepare findings
